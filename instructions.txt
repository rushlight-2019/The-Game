The Game  

----------------------------
INI file
SizeCell: 15 Default, my sreen uses 20. Works for my resolution which is 1680x1050. 

To change SizeCell, open game, quit.  Edit TheGame.ini.

-------------------------------------------------------------


Game used a 200ms timing loop.  My Win 7 uses 40-50 ms per loop. My Win 10 about 120-170ms


Game Loop
You move (up left/right), objects move, you move down.  Tick time delay to 200ms, repeat.

----------------------------------------------------------------------------------------
Game: Keyboad only. Mouse can be used to activate C and Esc.
C = Cancel Game back to start Menu
ESC = Will kill you if you get stuck. Some level have rooms you can't leave

Starts at level 1. 

On start it ask for a Code Key to start 
---------	

Diamond:  You must pick up all to go to next level. Jump on and press down.


Happy face is you.  You only move left or right. By the way you move you will go up. Nothing under you, you will drop (fast delay).


Lift  light blue block: You can ride it up only, but get off somewhere on the way up or splat. Best: Start getting off one cell early. Timing.
	Lift will return to start when you get off.

Vertical moving blocks. Triangle:  You can ride down but not up. Don't run into them, it deadly.  

Horizonal moving blocks. + - :  Deadly if you run to them or them into you.  
                                You can jump on top and when they move a way you will drop.

Hidden area. You need the torch to see. There moving hidden horizonal blocks.

~~ you can ride down. They DON'T go back up.


Missile: You seeking. Star:  Turn off with the switch. Get on top of switch press down.  Green or Red star.
Key, Torch which you should pickup. (Get on top press down)

Hidden pieces (need torch) and a Door (need key).


--------------------------------------------------------
Edit screen   

The white square with the red + is the cursor.

Press Colors, Objects or Control for different buttons below them.  

Can move with arrow key or mouse left click 

To add text press button.  

To add it to the map press space bar or right click. 
To pick up from map press C or M (NEW)

There a Repeat button, to replace a large number of squares.  Uses the arrors keys to work.  

If you add YOU, Missile, Missile Switch, Key or Torch.  The button will not show. To remove this put something in there place, the buttons will come back.

Quit  Will warn if level been changed

New - Will find first level number not used, if NO then if will find next.

Test Level - Will NOT automatical save the level. (on the to do list)
	It will run the game for 'Edit level' only.  Endless retries.

Does not test for more then one you or missle.  Uses the last one loaded.  Loads starts from bottom up.
